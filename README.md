# Marketing Campaign Analysis

This project analyzes marketing campaign performance to identify the most effective campaigns, channels, and revenue-generating strategies.

## Project Overview
The goal of this project is to evaluate marketing campaign performance by analyzing key metrics such as revenue, conversions, clicks, and campaign effectiveness.

## Tools Used
- SQL
- Microsoft Excel
- Power BI

## Project Workflow
1. Data cleaning and preparation in Excel
2. Data analysis using SQL queries
3. Data visualization using Power BI dashboard

## Business Questions Solved
- Which marketing campaign generated the highest revenue?
- Which marketing channel performed the best?
- What is the conversion rate for each campaign?
- Which campaigns produced the most conversions?

## Dashboard Features
- Total Revenue KPI
- Total Conversions KPI
- Campaign Performance Analysis
- Channel-wise Revenue Analysis
- Campaign Comparison
